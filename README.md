# 💾 Projeto DAO com JDBC - Curso Java

[![NPM](https://img.shields.io/npm/l/react)](https://github.com/brunoproencadev/demo-dao-jdbc/blob/main/LICENSE) 

## 📌 Sobre o Projeto

Este é o meu primeiro projeto prático de acesso a Banco de Dados desenvolvido durante o curso de Java ministrado pelo Dr. Nelio Alves. 

O objetivo principal deste projeto é demonstrar como uma aplicação Java se comunica com um banco de dados relacional (como o MySQL) utilizando o **JDBC** (Java Database Connectivity) puro, sem o uso de frameworks complexos. 

Para manter o código organizado e profissional, foi utilizado o padrão de projeto **DAO (Data Access Object)**. De forma simples, esse padrão serve para separar a parte do código que "conversa" com o banco de dados da parte do código que faz o programa funcionar.

## ⚙️ Funcionalidades

O sistema é focado em realizar o famoso **CRUD** (Create, Read, Update, Delete), que são as 4 operações básicas de qualquer sistema com banco de dados:

- **Create (Criar):** Inserir novos registros no banco de dados.
- **Read (Ler):** Buscar registros (seja buscando todos de uma vez ou buscando por um identificador específico).
- **Update (Atualizar):** Modificar as informações de um registro que já existe.
- **Delete (Deletar):** Apagar um registro do banco de dados.

## 🛠️ Tecnologias Utilizadas

- **Java:** Linguagem principal do projeto.
- **JDBC:** Biblioteca padrão do Java para conectar e interagir com o banco de dados.
- **MySQL:** Banco de dados relacional utilizado para guardar as informações.
- **IDE:** Eclipse.
